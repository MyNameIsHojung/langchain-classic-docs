Module path

langchain_classic.hub

Summary

Push and pull prompts and runnables from the LangChain Hub.

When to use

  • Pull/push prompts or runnables via the Hub.

Typical workflow

  • hub.pull(...) / hub.push(...).

Migration notes

  • Prefer the langsmith hub client.

What it provides

  • push serializes a LangChain object and uploads it to the hub.
  • pull fetches an object from the hub and deserializes it.
  • Uses langsmith.Client when available, with a fallback to the legacy langchainhub client.

Notable exports

  • push, pull.

Notes and shims

  • Requires langsmith or langchainhub to be installed. The preferred path is langsmith.
  • API key and api_url can be provided explicitly or picked up from environment configuration.

Examples

Beginner

from langchain_classic import hub

prompt = hub.pull("owner/prompt_name")

Intermediate

from langchain_classic.runnables.hub import HubRunnable

runnable = HubRunnable("owner/prompt_name")
result = runnable.invoke({"input": "hi"})

Advanced

from langchain_classic import hub

hub.push(
    "owner/prompt_name",
    object=prompt,
    tags=["classic", "docs"],
)
