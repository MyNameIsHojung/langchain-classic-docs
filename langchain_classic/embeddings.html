Module path

langchain_classic.embeddings

Summary

Embedding model wrappers, cache helpers, and provider shims.

When to use

  • Unified init_embeddings or cache-backed embeddings.

Typical workflow

  • init_embeddings(...) then optional CacheBackedEmbeddings.

Migration notes

  • Prefer provider packages; keep cache via storage.

What it provides

  • init_embeddings builds provider-specific embeddings from a unified model string.
  • CacheBackedEmbeddings wraps an embedder with a key-value cache.
  • Re-exports many provider embeddings from langchain_community.

Notable exports

  • init_embeddings.
  • CacheBackedEmbeddings.
  • HypotheticalDocumentEmbedder (from chains.hyde).
  • Provider embeddings such as OpenAIEmbeddings, CohereEmbeddings, HuggingFaceEmbeddings, etc. via dynamic import.

Notes and shims

  • Supported providers in init_embeddings include openai, azure_openai, bedrock, cohere, google_genai, google_vertexai, huggingface, mistralai, and ollama.
  • Most provider classes are shims to langchain_community and require installing the integration packages.

Examples

Beginner

from langchain_classic.embeddings import init_embeddings

embedder = init_embeddings("openai:text-embedding-3-small")
embedder.embed_query("hello")

Intermediate

from langchain_classic.embeddings import CacheBackedEmbeddings
from langchain_classic.storage import LocalFileStore
from langchain_openai import OpenAIEmbeddings

store = LocalFileStore("./embedding_cache")
underlying = OpenAIEmbeddings()
embedder = CacheBackedEmbeddings.from_bytes_store(
    underlying, store, namespace=underlying.model
)

Advanced

from langchain_classic.chains.hyde.base import HypotheticalDocumentEmbedder
from langchain_classic.embeddings import init_embeddings
from langchain_core.language_models import FakeListLLM

llm = FakeListLLM(responses=["hypothetical doc"])
base = init_embeddings("openai:text-embedding-3-small")
hyde = HypotheticalDocumentEmbedder.from_llm(llm, base)
hyde.embed_query("What is LCEL?")
