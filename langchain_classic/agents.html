Module path

langchain_classic.agents

Summary

Classic agent framework for tool-using workflows and action selection.

When to use

  • Legacy agent executors (ZeroShot/MRKL/ReAct/Structured/XML).
  • Tool-driven workflows in the pre-0.2 API.

Typical workflow

  • Define tools (Tool or @tool) and choose an agent factory.
  • Create AgentExecutor/initialize_agent and invoke with input.

Migration notes

  • Prefer LangGraph or LCEL tool-calling for new systems.

What it provides

  • Defines core agent interfaces and executors that loop over actions and tool calls.
  • Provides classic agent types such as ZeroShot, MRKL, ReAct, conversational, structured, and XML agents.
  • Includes factory helpers for OpenAI functions, tool calling, JSON/structured chat, and vectorstore agents.
  • Re-exports Tool and the tool decorator from langchain_core.tools for convenience.

Notable exports

  • Core classes: Agent, AgentExecutor, AgentExecutorIterator, AgentOutputParser.
  • Base types: BaseSingleActionAgent, BaseMultiActionAgent, LLMSingleActionAgent, AgentType.
  • Agent flavors: ZeroShotAgent, MRKLChain, ReActChain, ReActTextWorldAgent, ConversationalAgent, ConversationalChatAgent, StructuredChatAgent, XMLAgent.
  • OpenAI flavors: OpenAIFunctionsAgent, OpenAIMultiFunctionsAgent.
  • Factories: initialize_agent, load_agent, create_openai_functions_agent, create_openai_tools_agent, create_tool_calling_agent.
  • Factories (classic): create_react_agent, create_structured_chat_agent, create_json_chat_agent, create_self_ask_with_search_agent.
  • Vectorstore helpers: create_vectorstore_agent, create_vectorstore_router_agent.
  • Tool helpers: Tool, tool.

Notes and shims

  • Some toolkits are resolved dynamically from langchain_community (for example create_sql_agent, create_openapi_agent, load_tools). Install langchain-community for those imports.
  • create_csv_agent, create_pandas_dataframe_agent, create_spark_dataframe_agent, and create_xorbits_agent raise ImportError and are moved to langchain_experimental.

Examples

Beginner

from langchain_classic.agents import AgentType, initialize_agent
from langchain_core.language_models import FakeListLLM
from langchain_core.tools import Tool

llm = FakeListLLM(responses=["ok"])
tools = [Tool(name="echo", func=lambda x: x, description="Echo input")]
agent = initialize_agent(tools, llm, agent=AgentType.ZERO_SHOT_REACT_DESCRIPTION)
agent.invoke({"input": "hello"})

Intermediate

from langchain_classic.agents import AgentType, initialize_agent
from langchain_core.language_models import FakeListLLM
from langchain_core.tools import Tool

llm = FakeListLLM(responses=["ok"])
tools = [Tool(name="echo", func=lambda x: x, description="Echo input")]
agent = initialize_agent(
    tools, llm, agent=AgentType.ZERO_SHOT_REACT_DESCRIPTION, return_intermediate_steps=True
)
result = agent.invoke({"input": "hi"})
print(result["intermediate_steps"])

Advanced

from langchain_openai import ChatOpenAI
from langchain_classic.agents import create_tool_calling_agent, AgentExecutor
from langchain_core.tools import tool

@tool
def add(a: int, b: int) -> int:
    return a + b

llm = ChatOpenAI()
agent = create_tool_calling_agent(llm, [add])
executor = AgentExecutor(agent=agent, tools=[add])
executor.invoke({"input": "Add 2 and 3"})
