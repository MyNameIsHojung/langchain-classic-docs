Module path

langchain_classic.retrievers

Summary

Retrievers that return documents for a text query.

When to use

  • Provide documents for RAG.

Typical workflow

  • Start with a vectorstore retriever, then wrap for multi-query/compression.

Migration notes

  • Import retrievers from langchain_community for new integrations.

What it provides

  • Classic retrievers such as multi-query, multi-vector, parent-document, and time-weighted retrievers.
  • Contextual compression and ensemble retrievers for improving retrieval quality.
  • Dynamic import shims for many external-system retrievers in langchain_community.

Notable exports

  • ContextualCompressionRetriever, EnsembleRetriever, MergerRetriever.
  • MultiQueryRetriever, MultiVectorRetriever, ParentDocumentRetriever.
  • SelfQueryRetriever, RePhraseQueryRetriever, TimeWeightedVectorStoreRetriever.
  • Community retrievers such as BM25Retriever, AmazonKendraRetriever, PineconeHybridSearchRetriever, etc. (dynamic imports).

Notes and shims

  • Most external retrievers are provided by langchain_community and require provider-specific dependencies.

Examples

Beginner

from langchain_community.retrievers import BM25Retriever
from langchain_core.documents import Document

texts = ["cats are great", "dogs like walks", "cats purr"]
docs = [Document(page_content=t) for t in texts]
retriever = BM25Retriever.from_documents(docs)
retriever.invoke("cats")

Intermediate

from langchain_classic.retrievers import MultiQueryRetriever
from langchain_community.retrievers import BM25Retriever
from langchain_core.documents import Document
from langchain_core.language_models import FakeListLLM

docs = [Document(page_content="cats"), Document(page_content="dogs")]
base = BM25Retriever.from_documents(docs)
llm = FakeListLLM(responses=["cat care
cat behavior
cat food"])
retriever = MultiQueryRetriever.from_llm(retriever=base, llm=llm)
retriever.invoke("How to care for a cat?")

Advanced

from langchain_classic.retrievers import SelfQueryRetriever

retriever = SelfQueryRetriever.from_llm(
    llm=llm,
    vectorstore=vectorstore,
    document_contents="product reviews",
    metadata_field_info=metadata_field_info,
)
retriever.invoke("Affordable shoes under $100")
