Module path

langchain_classic.callbacks

Summary

Callback handlers and tracing helpers for observing runs.

When to use

  • Tracing/logging/streaming for chains or agents.

Typical workflow

  • Pass callbacks=[...] to invoke/astream; use collect_runs.

Migration notes

  • Import integration callbacks from langchain_community.

What it provides

  • Standard handlers for stdout and file logging.
  • Streaming helpers for async iterators and final-only stdout.
  • Tracing helpers with LangChainTracer plus tracing_v2_enabled and collect_runs.
  • Proxy to many integration-specific handlers via langchain_community.

Notable exports

  • StdOutCallbackHandler, StreamingStdOutCallbackHandler, FileCallbackHandler.
  • AsyncIteratorCallbackHandler, FinalStreamingStdOutCallbackHandler.
  • LangChainTracer, collect_runs, tracing_v2_enabled.
  • Community handlers such as WandbCallbackHandler, MlflowCallbackHandler, OpenAICallbackHandler, etc. (dynamic imports).

Notes and shims

  • Most provider-specific callback handlers are imported dynamically from langchain_community and will warn about deprecation when used from langchain_classic.

Examples

Beginner

from langchain_classic.callbacks import StdOutCallbackHandler
from langchain_classic.chains import LLMChain
from langchain_core.language_models import FakeListLLM
from langchain_core.prompts import PromptTemplate

llm = FakeListLLM(responses=["hello"])
prompt = PromptTemplate.from_template("Say {thing}")
chain = LLMChain(llm=llm, prompt=prompt)
chain.invoke({"thing": "hi"}, callbacks=[StdOutCallbackHandler()])

Intermediate

from langchain_classic.callbacks import collect_runs

with collect_runs() as cb:
    chain.invoke({"thing": "hi"})
print(len(cb.traced_runs))

Advanced

from langchain_classic.callbacks import LangChainTracer

tracer = LangChainTracer(project_name="classic-docs")
chain.invoke({"thing": "hi"}, callbacks=[tracer])
