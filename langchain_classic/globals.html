Module path

langchain_classic.globals

Summary

Global debug, verbose, and cache configuration hooks.

When to use

  • Set debug/verbose/cache globally.

Typical workflow

  • Call set_debug/set_verbose/set_llm_cache at startup.

Migration notes

  • Prefer per-run config in new code.

What it provides

  • Re-exports the global setters and getters from langchain_core.globals.

Notable exports

  • get_debug, set_debug.
  • get_verbose, set_verbose.
  • get_llm_cache, set_llm_cache.

Notes and shims

  • These settings apply process-wide and affect classic chains, agents, and runnables.

Examples

Beginner

from langchain_classic.globals import set_debug

set_debug(True)

Intermediate

from langchain_classic.globals import set_verbose, get_verbose

set_verbose(True)
print(get_verbose())

Advanced

from langchain_classic.globals import set_llm_cache
from langchain_community.cache import InMemoryCache

set_llm_cache(InMemoryCache())
