Module path

langchain_classic.smith

Summary

LangSmith evaluation runner utilities for datasets.

When to use

  • Run dataset evaluation and capture results.

Typical workflow

  • Client + RunEvalConfig + run_on_dataset.

Migration notes

  • Use the langsmith SDK directly for advanced cases.

What it provides

  • run_on_dataset and arun_on_dataset evaluate a chain or agent over a dataset.
  • RunEvalConfig configures which evaluators run and how results are scored.
  • Integrates with langsmith.Client for dataset access and logging.

Notable exports

  • RunEvalConfig, run_on_dataset, arun_on_dataset.

Notes and shims

  • The module name is langchain_classic.smith (not langchain_classic.langsmith).
  • Requires the langsmith package for client access.

Examples

Beginner

from langsmith import Client
from langchain_classic.smith import run_on_dataset

client = Client()
run_on_dataset(client, "my_dataset", construct_chain)

Intermediate

from langchain_classic.smith import RunEvalConfig

config = RunEvalConfig(evaluators=["qa", RunEvalConfig.Criteria("helpfulness")])

Advanced

from langsmith import Client
from langchain_classic.smith import arun_on_dataset
