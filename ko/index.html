About this documentation

  • langchain_classic의 로컬 패키지 소스에서 빌드했습니다.
  • langchain_classic은 주로 langchain_core와 langchain_community의 심볼을 재내보내는 compatibility shim입니다.
  • Dynamic imports는 interactive 환경 밖에서 LangChain deprecation warnings를 발생시킵니다.
  • 새 프로젝트에서는 modern packages(langchain_core, langchain_community, langchain_text_splitters)와 LCEL workflows를 우선 사용하세요.

How to use

  • 왼쪽 모듈을 탐색해 각 reference page를 여세요.
  • 각 페이지는 로컬 패키지 소스의 exports, shims, notes를 요약합니다.

Overview

  • langchain_classic은 프로젝트가 split packages로 이동하는 동안 legacy LangChain API surface를 유지합니다.
  • Core packages는 이제 langchain_core, langchain_community, 그리고 langchain_text_splitters에 있습니다.
  • 대부분 모듈은 dynamic import shims이며, 일부 classic implementations는 backward compatibility를 위해 남아 있습니다.

legacy API 또는 아직 마이그레이션되지 않은 classic chains/agents가 필요할 때 langchain_classic을 사용하세요. 새 프로젝트에는 LCEL과 modern packages를 권장합니다.

Quickstart

Classic chain (legacy API)

from langchain_core.language_models import FakeListLLM
from langchain_core.prompts import PromptTemplate
from langchain_classic.chains import LLMChain

llm = FakeListLLM(responses=["hello"])
prompt = PromptTemplate.from_template("Say {thing}")
chain = LLMChain(llm=llm, prompt=prompt)

result = chain.invoke({"thing": "hi"})
print(result["text"])

LCEL equivalent (recommended)

from langchain_core.language_models import FakeListLLM
from langchain_core.output_parsers import StrOutputParser
from langchain_core.prompts import PromptTemplate

prompt = PromptTemplate.from_template("Say {thing}")
llm = FakeListLLM(responses=["hello"])
chain = prompt | llm | StrOutputParser()

print(chain.invoke({"thing": "hi"}))

Using the Hub

from langchain_classic import hub

prompt = hub.pull("owner/prompt_name")

Package layout (high level)

  • Top-level helpers: globals, hub, env, load, text_splitter, cache.
  • Classic implementations: agents, chains, memory, output_parsers, retrievers.
  • Community-backed shims: chat_models, embeddings, llms, tools, utilities, vectorstores.

Deprecations and migration

  • from langchain_classic import OpenAI → from langchain_community.llms import OpenAI
  • from langchain_classic import FAISS → from langchain_community.vectorstores import FAISS
  • from langchain_classic import PromptTemplate → from langchain_core.prompts import PromptTemplate
  • from langchain_classic.text_splitter import RecursiveCharacterTextSplitter → from langchain_text_splitters import RecursiveCharacterTextSplitter

LLMChain과 classic agents는 compatibility를 위해 유지되지만, 새 작업에는 LCEL과 LangGraph를 권장합니다.

Troubleshooting

  • ModuleNotFoundError: langchain_community → integrations를 위해 langchain-community를 설치하세요.
  • ImportError: moved to langchain_experimental → langchain_experimental과 sandbox tools를 사용하세요.
  • Unexpected deprecation warnings → langchain_core / langchain_community에서 직접 import하는 것을 권장합니다.
