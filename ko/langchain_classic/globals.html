Module path

langchain_classic.globals

Summary

전역 debug/verbose/캐시 설정 훅입니다.

When to use

  • debug/verbose/캐시를 전역으로 설정할 때.

Typical workflow

  • 시작 시 set_debug/set_verbose/set_llm_cache를 호출합니다.

Migration notes

  • 신규 코드에서는 실행별 config를 우선 사용하세요.

What it provides

  • langchain_core.globals의 전역 setter/getter를 재내보냅니다.

Notable exports

  • get_debug, set_debug.
  • get_verbose, set_verbose.
  • get_llm_cache, set_llm_cache.

Notes and shims

  • 이 설정은 프로세스 전체에 적용되며 클래식 체인, 에이전트, 러너블에 영향을 줍니다.

Examples

Beginner

from langchain_classic.globals import set_debug

set_debug(True)

Intermediate

from langchain_classic.globals import set_verbose, get_verbose

set_verbose(True)
print(get_verbose())

Advanced

from langchain_classic.globals import set_llm_cache
from langchain_community.cache import InMemoryCache

set_llm_cache(InMemoryCache())
