Module path

langchain_classic.memory

Summary

체인과 에이전트를 위한 상태 보존 메모리 구현입니다.

When to use

  • 실행 간 대화 상태를 유지할 때.

Typical workflow

  • 메모리 클래스를 선택해 체인/에이전트에 연결합니다.

Migration notes

  • 신규 앱에서는 외부 메모리 스토어를 고려하세요.

What it provides

  • 대화 컨텍스트를 위한 버퍼/Summary 메모리.
  • 더 긴 히스토리를 위한 토큰 버퍼 및 vectorstore 기반 메모리.
  • 결합/읽기 전용 메모리 어댑터와 엔터티 스토어 기반 엔터티 메모리.
  • 채팅 메시지 히스토리 구현은 langchain_community에서 동적으로 임포트됩니다.

Notable exports

  • ConversationBufferMemory, ConversationStringBufferMemory, ConversationBufferWindowMemory.
  • ConversationSummaryMemory, ConversationSummaryBufferMemory, ConversationTokenBufferMemory.
  • ConversationVectorStoreTokenBufferMemory, VectorStoreRetrieverMemory.
  • CombinedMemory, ReadOnlySharedMemory, SimpleMemory.
  • ConversationEntityMemory, InMemoryEntityStore, RedisEntityStore, SQLiteEntityStore, UpstashRedisEntityStore.
  • RedisChatMessageHistory, SQLChatMessageHistory 등 채팅 메시지 히스토리(동적 임포트).

Notes and shims

  • 커뮤니티 채팅 히스토리 구현에는 langchain-community와 해당 드라이버가 필요합니다.

Examples

Beginner

from langchain_classic.memory import ConversationBufferMemory

memory = ConversationBufferMemory(return_messages=True)

Intermediate

from langchain_classic.memory import ConversationSummaryBufferMemory
from langchain_core.language_models import FakeListLLM

llm = FakeListLLM(responses=["summary"])
memory = ConversationSummaryBufferMemory(llm=llm, max_token_limit=50)

Advanced

from langchain_classic.memory import VectorStoreRetrieverMemory

memory = VectorStoreRetrieverMemory(retriever=retriever)
