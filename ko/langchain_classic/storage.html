Module path

langchain_classic.storage

Summary

캐싱과 영속화를 위한 key-value 스토어와 storage 헬퍼입니다.

When to use

  • KV 스토어에 임베딩/문서를 캐시할 때.

Typical workflow

  • InMemoryStore 또는 LocalFileStore/EncoderBackedStore를 사용합니다.

Migration notes

  • 신규 코드는 langchain_core.stores를 우선 사용하세요.

What it provides

  • bytes 및 임의 값용 인메모리 스토어.
  • 파일 시스템 기반 byte 스토어와 encoder-backed 스토어 래퍼.
  • LangChain docstore와 LC store를 구축하는 헬퍼.
  • langchain_community의 Redis/Upstash 스토어에 대한 동적 임포트.

Notable exports

  • InMemoryStore, InMemoryByteStore, InvalidKeyException.
  • LocalFileStore, EncoderBackedStore.
  • create_kv_docstore, create_lc_store.
  • RedisStore, UpstashRedisStore, UpstashRedisByteStore (동적 임포트).

Notes and shims

  • Redis와 Upstash 스토어는 langchain-community와 해당 클라이언트 라이브러리가 필요합니다.

Examples

Beginner

from langchain_classic.storage import InMemoryStore

store = InMemoryStore()
store.mset([("key", {"value": 1})])

Intermediate

from langchain_classic.storage import LocalFileStore

store = LocalFileStore("./cache_dir")

Advanced

from langchain_classic.storage import EncoderBackedStore, LocalFileStore
import json

base = LocalFileStore("./cache_dir")
store = EncoderBackedStore(
    base,
    key_encoder=lambda k: k,
    value_serializer=lambda v: json.dumps(v).encode(),
    value_deserializer=lambda b: json.loads(b.decode()),
)
