Module path

langchain_classic.smith

Summary

데이터셋용 LangSmith 평가 러너 유틸리티입니다.

When to use

  • 데이터셋 평가를 실행하고 결과를 수집할 때.

Typical workflow

  • Client + RunEvalConfig + run_on_dataset.

Migration notes

  • Advanced 사례에는 langsmith SDK를 직접 사용하세요.

What it provides

  • run_on_dataset과 arun_on_dataset은 데이터셋에서 체인/에이전트를 평가합니다.
  • RunEvalConfig는 어떤 평가기를 실행할지와 결과 점수 산정을 설정합니다.
  • 데이터셋 접근과 로깅을 위해 langsmith.Client와 통합됩니다.

Notable exports

  • RunEvalConfig, run_on_dataset, arun_on_dataset.

Notes and shims

  • 모듈 이름은 langchain_classic.smith이며 langchain_classic.langsmith가 아닙니다.
  • 클라이언트 사용을 위해 langsmith 패키지가 필요합니다.

Examples

Beginner

from langsmith import Client
from langchain_classic.smith import run_on_dataset

client = Client()
run_on_dataset(client, "my_dataset", construct_chain)

Intermediate

from langchain_classic.smith import RunEvalConfig

config = RunEvalConfig(evaluators=["qa", RunEvalConfig.Criteria("helpfulness")])

Advanced

from langsmith import Client
from langchain_classic.smith import arun_on_dataset

client = Client()
# await arun_on_dataset(client, "my_dataset", construct_chain)
