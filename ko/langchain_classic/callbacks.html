Module path

langchain_classic.callbacks

Summary

실행을 관찰하기 위한 콜백 핸들러와 트레이싱 헬퍼입니다.

When to use

  • 체인 또는 에이전트의 tracing/logging/streaming이 필요할 때.

Typical workflow

  • invoke/astream에 callbacks=[...]를 전달하고 collect_runs를 사용합니다.

Migration notes

  • 통합 콜백은 langchain_community에서 직접 임포트하세요.

What it provides

  • stdout 및 파일 로깅용 표준 핸들러.
  • async iterator와 final-only stdout용 스트리밍 헬퍼.
  • LangChainTracer와 tracing_v2_enabled, collect_runs를 통한 트레이싱 헬퍼.
  • langchain_community를 통해 많은 통합 전용 핸들러로 프록시.

Notable exports

  • StdOutCallbackHandler, StreamingStdOutCallbackHandler, FileCallbackHandler.
  • AsyncIteratorCallbackHandler, FinalStreamingStdOutCallbackHandler.
  • LangChainTracer, collect_runs, tracing_v2_enabled.
  • WandbCallbackHandler, MlflowCallbackHandler, OpenAICallbackHandler 등 커뮤니티 핸들러 (동적 임포트).

Notes and shims

  • 대부분의 프로바이더 전용 콜백 핸들러는 langchain_community에서 동적으로 임포트되며, langchain_classic 경로에서 사용할 때 deprecation 경고가 발생합니다.

Examples

Beginner

from langchain_classic.callbacks import StdOutCallbackHandler
from langchain_classic.chains import LLMChain
from langchain_core.language_models import FakeListLLM
from langchain_core.prompts import PromptTemplate

llm = FakeListLLM(responses=["hello"])
prompt = PromptTemplate.from_template("Say {thing}")
chain = LLMChain(llm=llm, prompt=prompt)
chain.invoke({"thing": "hi"}, callbacks=[StdOutCallbackHandler()])

Intermediate

from langchain_classic.callbacks import collect_runs

with collect_runs() as cb:
    chain.invoke({"thing": "hi"})
print(len(cb.traced_runs))

Advanced

from langchain_classic.callbacks import LangChainTracer

tracer = LangChainTracer(project_name="classic-docs")
chain.invoke({"thing": "hi"}, callbacks=[tracer])
