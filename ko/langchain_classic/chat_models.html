Module path

langchain_classic.chat_models

Summary

채팅 모델 통합을 위한 호환성 레이어와 통합 초기화 도구입니다.

When to use

  • 레거시 임포트 경로를 써야 하거나 통합 init_chat_model이 필요할 때.

Typical workflow

  • init_chat_model("provider:model")을 사용하거나 langchain_community.chat_models에서 임포트합니다.

Migration notes

  • langchain_community 또는 프로바이더 패키지에서 직접 임포트하세요.

What it provides

  • 인터랙티브 환경 밖에서는 deprecation 경고와 함께 langchain_community의 채팅 모델을 재내보냅니다.
  • 프로바이더에 독립적인 초기화와 런타임 구성 변경을 위한 init_chat_model을 제공합니다.
  • 기본 모듈에 langchain_core의 BaseChatModel과 SimpleChatModel을 포함합니다.

Notable exports

  • init_chat_model.
  • ChatOpenAI, AzureChatOpenAI, ChatAnthropic, BedrockChat, ChatVertexAI, ChatCohere, ChatOllama, ChatGooglePalm, ChatDatabricks.
  • FakeListChatModel, HumanInputChatModel, PromptLayerChatOpenAI.
  • 그 외 다수의 프로바이더 전용 채팅 모델이 langchain_community.chat_models에서 동적으로 노출됩니다.

Notes and shims

  • langchain_classic에서 채팅 모델을 임포트하면 인터랙티브 환경 밖에서 LangChainDeprecationWarning이 발생합니다.
  • init_chat_model 사용을 위해 프로바이더 통합 패키지(예: langchain-openai)가 설치되어 있어야 합니다.

Examples

Beginner

from langchain_classic.chat_models import init_chat_model

model = init_chat_model("openai:gpt-3.5-turbo")
model.invoke("Hello")

Intermediate

from langchain_classic.chat_models import init_chat_model

model = init_chat_model(configurable_fields="any")
model.invoke(
    "Hello",
    config={"configurable": {"model": "openai:gpt-4o-mini"}},
)

Advanced

from langchain_community.chat_models import ChatOpenAI
from langchain_core.messages import HumanMessage

model = ChatOpenAI()
model.invoke([HumanMessage(content="Hello")])
