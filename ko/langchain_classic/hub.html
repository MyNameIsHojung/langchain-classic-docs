Module path

langchain_classic.hub

Summary

LangChain Hub에서 프롬프트와 runnable을 push/pull합니다.

When to use

  • Hub를 통해 프롬프트 또는 runnable을 pull/push할 때.

Typical workflow

  • hub.pull(...) / hub.push(...).

Migration notes

  • langsmith hub client를 우선 사용하세요.

What it provides

  • push는 LangChain 객체를 직렬화해 hub에 업로드합니다.
  • pull은 hub에서 객체를 가져와 역직렬화합니다.
  • 가능하면 langsmith.Client를 사용하고, 없으면 레거시 langchainhub client로 폴백합니다.

Notable exports

  • push, pull.

Notes and shims

  • langsmith 또는 langchainhub 설치가 필요하며, 권장 경로는 langsmith입니다.
  • API key와 api_url은 직접 지정하거나 환경 설정에서 가져올 수 있습니다.

Examples

Beginner

from langchain_classic import hub

prompt = hub.pull("owner/prompt_name")

Intermediate

from langchain_classic.runnables.hub import HubRunnable

runnable = HubRunnable("owner/prompt_name")
result = runnable.invoke({"input": "hi"})

Advanced

from langchain_classic import hub

hub.push(
    "owner/prompt_name",
    object=prompt,
    tags=["classic", "docs"],
)
