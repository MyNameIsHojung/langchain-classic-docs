Module path

langchain_classic.output_parsers

Summary

LLM 출력을 구조화 데이터로 변환하는 파서입니다.

When to use

  • LLM 출력을 구조화 데이터로 파싱할 때.

Typical workflow

  • 포맷 지침을 포함한 뒤 응답을 파싱합니다.

Migration notes

  • langchain_core.output_parsers를 우선 사용하세요.

What it provides

  • List, CSV, Markdown, numbered list, XML, YAML, Pydantic 기반 파서.
  • Regex 기반 파서와 구조화 출력 스키마.
  • 잘못된 출력을 보정하는 retry/fixing 파서.
  • tool-call JSON용 OpenAI tool 파서.

Notable exports

  • ListOutputParser, CommaSeparatedListOutputParser, MarkdownListOutputParser, NumberedListOutputParser.
  • PydanticOutputParser, XMLOutputParser, YamlOutputParser.
  • StructuredOutputParser, ResponseSchema.
  • RegexParser, RegexDictParser, BooleanOutputParser, DatetimeOutputParser, EnumOutputParser.
  • OutputFixingParser, RetryOutputParser, RetryWithErrorOutputParser, CombiningOutputParser.
  • JsonOutputToolsParser, JsonOutputKeyToolsParser, PydanticToolsParser.

Notes and shims

  • GuardrailsOutputParser는 가능할 때 langchain_community.output_parsers.rail_parser에서 해결됩니다.

Examples

Beginner

from langchain_classic.output_parsers import ResponseSchema, StructuredOutputParser

schemas = [ResponseSchema(name="answer", description="Final answer")]
parser = StructuredOutputParser.from_response_schemas(schemas)
format_instructions = parser.get_format_instructions()

Intermediate

from pydantic import BaseModel, Field
from langchain_classic.output_parsers import PydanticOutputParser

class Person(BaseModel):
    name: str = Field(...)
    age: int = Field(...)

parser = PydanticOutputParser(pydantic_object=Person)

Advanced

from langchain_classic.output_parsers import OutputFixingParser, PydanticOutputParser
from langchain_openai import ChatOpenAI

parser = PydanticOutputParser(pydantic_object=Person)
fixer = OutputFixingParser.from_llm(parser=parser, llm=ChatOpenAI())
